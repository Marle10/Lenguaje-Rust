# Calculadora en Rust

Una **calculadora de línea de comandos** escrita en **Rust**, que permite realizar operaciones básicas como **suma, resta, multiplicación y división**, con validación de errores y un modo interactivo tipo REPL.

---

## Características principales

- ✅ Soporta las operaciones: `+`, `-`, `*`, `/`
- ✅ Permite introducir expresiones con o sin espacios (`3 + 4` o `3+4`)
- ✅ Detecta errores de entrada y división por cero
- ✅ Modo interactivo: el usuario puede realizar varias operaciones seguidas
- ✅ Opción de salida con el comando `salir`
- ✅ Código modular, documentado y fácil de extender
